FastShip HU - Thư mục Public

Thư mục này đã được cấu hình thành công để truy cập trực tiếp!

Bạn có thể đặt các file tĩnh như hình ảnh, tài liệu PDF, tệp CSS, JavaScript và các tài nguyên khác vào thư mục này.

Ví dụ về cách truy cập:

Để truy cập file test.txt, hãy dùng URL: /test.txt

Để truy cập hình ảnh logo.png (nếu có), hãy dùng URL: /logo.png

Với cấu hình này, bạn không cần phải tạo route đặc biệt để truy cập vào từng file tĩnh. Express sẽ tự động phục vụ tất cả file từ thư mục public.
